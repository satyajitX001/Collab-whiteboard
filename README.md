# Whiteboard App

A real-time collaborative whiteboard application built with Next.js, Node.js, and Socket.io.

## Prerequisites

- Node.js installed

## Getting Started

### 1. Start the Backend Server

Open a terminal and run:

```bash
cd server
npm install
npm start
```

The server will run on `http://localhost:3001`.

### 2. Start the Frontend Client

Open a new terminal window and run:

```bash
cd client
npm install
npm run dev
```

The client will run on `http://localhost:3000`.

## Features

- Real-time drawing sync across multiple windows/users.
- Color picker.
- Clear board functionality.
